== ABOUT "We Love WordPress" ==

The "We Love WordPress" design was developed by Odyssey for the Sandbox Designs Competition.

A theme featuring the movement known as We Love WordPress, http://www.flickr.com/photos/cafelog/sets/72157594207501792/, about WordPress in Japan, and the penguin character called as "Warp", which I created.


== USING THIS DESIGN ==

** Please see howtoinstall.txt for information on installing this design.

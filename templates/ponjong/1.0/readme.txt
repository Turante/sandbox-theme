== ABOUT PONJONG ==

The PONJONG design was developed by NURUDIN JAUHARI for the Sandbox Designs Competition.

This purpose of this design is simple and minimalistic, without images, so this design very light to load and of course more faster loading.

Basicly this design use dark color, to make strong enough to the reader.

For better IMG Float, add img class with right for float right or add class left for float left, example <img src="bla bla" class="left" />
